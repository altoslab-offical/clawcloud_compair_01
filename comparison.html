🦞 龙虾云 适合你吗？

  • 完全不需要电脑，手机即可使用
  • 付费立即可用，零技术门槛
  • 24/7 不间断运行，无需担心电脑关机
  • AI 费用包含在订阅内，无需额外准备
  • 适合中老年用户或技术小白
  • 数据存放于云端（东京 VPS）
  • 无法自行选择 AI 模型

💻 QClaw 适合你吗？

  • 可自备 API Key，灵活选择模型
  • 腾讯背书，适合有一定技术基础的用户
  • 电脑必须保持开机状态
  • 安装配置门槛较高
  • 目前仍在内测，需要抢码
  • 需自备 API Key 或腾讯额度
