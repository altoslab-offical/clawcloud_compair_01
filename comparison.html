🦞 龍蝦雲 適合你嗎？

  • 完全不需要電腦，手機即可使用
  • 付費立即可用，零技術門檻
  • 24/7 不間斷運行，無需擔心電腦關機
  • AI 費用包含在訂閱內，無需額外準備
  • 適合中老年用戶或技術小白
  • 數據存放於雲端（東京 VPS）
  • 無法自行選擇 AI 模型

💻 QClaw 適合你嗎？

  • 可自備 API Key，靈活選擇模型
  • 騰訊背書，適合有一定技術基礎的用戶
  • 電腦必須保持開機狀態
  • 安裝配置門檻較高
  • 目前仍在內測，需要搶碼
  • 需自備 API Key 或騰訊額度
